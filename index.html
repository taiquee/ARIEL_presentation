V1.0.0 — Fundação

Fevereiro 2026 · Status: COMPLETO E FUNCIONAL

A versão inicial do A.R.I.E.L estabelece toda a base do projeto: assistente CLI 100% local com reconhecimento de voz em português, síntese de fala e execução de ações reais no sistema operacional. Sem dependências de nuvem, sem rastreamento.

O ponto de partida de tudo — simples, funcional e 100% privado.

  • Assistente CLI 100% local
  • Reconhecimento de voz (Whisper + Google STT)
  • Síntese de fala (pyttsx3)
  • Processamento de intenções (NLP básico)
  • Execução de ações no sistema
  • Memória persistente em JSON
  • Suporte: Windows, Linux, macOS

V2.0.0 — Inteligência

Março 2026 · Status: COMPLETO E FUNCIONAL

A segunda versão eleva o A.R.I.E.L com integração ao GPT-4, interface gráfica via Tkinter e raciocínio em cadeia (Chain-of-Thought). O assistente deixa de ser apenas reativo e passa a raciocinar sobre os comandos.

Edge TTS substitui o pyttsx3, trazendo síntese de fala muito mais natural e de alta qualidade em português.

  • Interface gráfica (Tkinter)
  • Integração com OpenAI GPT-4
  • Raciocínio em cadeia (Chain-of-Thought)
  • Memória contextual melhorada
  • Edge TTS para síntese de fala
  • Suporte a múltiplos idiomas
  • Execução de ações avançadas

V3.0.0 — Experiência

Abril 2026 · Status: PRONTO PARA PRODUÇÃO

A versão mais ambiciosa. Interface HUD futurista construída com Electron + React, backend Flask com API REST e separação total entre modo texto e voz. Status dinâmico: Ouvindo → Processando → Pronto.

Chat inteligente com histórico completo, memória de curto e longo prazo, e instalador universal para qualquer plataforma. O sonho do Jarvis mais perto do que nunca.

  • Interface HUD futurista (Electron + React)
  • Backend Flask com API REST
  • Separação entre modo texto e voz
  • Chat inteligente com histórico
  • Edge TTS com fallback (gTTS, pyttsx3)
  • Status dinâmico do HUD em tempo real
  • Raciocínio avançado com contexto
  • Memória de curto e longo prazo
  • Instalador universal

Motivação e Visão

Estudante de Informática no Instituto Federal do Espírito Santo — Campus Serra, Arthur Taíque iniciou o projeto A.R.I.E.L em 21 de fevereiro de 2026 movido por uma visão clara: criar um assistente pessoal que vai além de um simples chatbot.

Inspirado por Tony Stark e o J.A.R.V.I.S do Universo Marvel, o objetivo é desenvolver algo local, privado e genuinamente inteligente — que aprende, executa e evolui. Em apenas dois meses, o projeto saiu de um CLI básico para uma interface HUD futurista com backend REST.

De V1 a V3, cada versão representa um salto real de capacidade e ambição. O A.R.I.E.L é a prova de que estudantes podem construir tecnologia de ponta — com dedicação, código aberto e a mentalidade certa.
